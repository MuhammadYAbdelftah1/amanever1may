## Image Guidelines

- **Format:** JPEG or PNG (JPEG preferred for photographs)
- **Quality:** High resolution, optimized for web
- **Aspect Ratio:** 4:3 or 16:9
- **File Size:** Aim for under 500KB per image (will be optimized by Next.js)
- **Style:** Professional, clean, modern healthcare imagery
- **Cultural Sensitivity:** Ensure images are appropriate for Saudi Arabian audience

## How to Add Images

1. Download images from Unsplash or Pexels (free license)
2. Rename them to match the filenames above
3. Place them in this directory (public/images/services/)
4. Next.js will automatically optimize them for web delivery
