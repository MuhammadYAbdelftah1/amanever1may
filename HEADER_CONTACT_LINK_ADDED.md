# إضافة رابط "اتصل بنا" في الهيدر ✅

## التغييرات المنفذة

### 1. تحديث ملف الهيدر
**الملف**: `components/layout/header.tsx`

تم إضافة رابط "اتصل بنا" / "Contact Us" / "ہم سے رابطہ کریں" في قائمة التنقل:

```typescript
const navLinks = [
  { href: `/${locale}`, label: t('home') },
  { href: `/${locale}/about`, label: t('about') },
  { href: `/${locale}/services`, label: t('services') },
  { href: `/${locale}/contact`, label: t('contact') }, // ✅ جديد
];
```

### 2. الترجمات الموجودة
الترجمات كانت موجودة بالفعل في ملفات الترجمة:

**العربية** (`messages/ar.json`):
```json
"contact": "تواصل معنا"
```

**الإنجليزية** (`messages/en.json`):
```json
"contact": "Contact Us"
```

**الأردية** (`messages/ur.json`):
```json
"contact": "ہم سے رابطہ کریں"
```

## النتيجة

### في الهيدر الآن:
1. **الصفحة الرئيسية** - الرئيسية / Home / ہوم
2. **من نحن** - من نحن / About Us / ہمارے بارے میں
3. **خدماتنا** - خدماتنا / Our Services / ہماری خدمات
4. **اتصل بنا** ✅ - تواصل معنا / Contact Us / ہم سے رابطہ کریں

### المزايا:
- ✅ يظهر في الهيدر الرئيسي (Desktop)
- ✅ يظهر في القائمة المنسدلة (Mobile)
- ✅ يدعم 3 لغات (عربي، إنجليزي، أردو)
- ✅ يتكيف مع تصميم الهيدر:
  - **الصفحة الرئيسية**: نص أبيض على خلفية شفافة
  - **الصفحات الأخرى**: نص بلون اللوجو (#5e8f8f) على خلفية بيضاء
- ✅ يحتوي على مؤشر نشط (Active indicator) عند التواجد في الصفحة
- ✅ تأثيرات Hover سلسة
