- **الأيقونة:** Store
- **الوصف:** للصيدليات والمراكز الطبية
- **الحقول المطلوبة:**

- **الحقول المطلوبة:**

  - ✅ البريد الإلكتروني (مطلوب)

  - ✅ **رقم الآيبان (IBAN)** (مطلوب) - يبدأ بـ SA
  - ✅ كلمة المرور (مطلوب)
  - ✅ تأكيد كلمة المرور (مطلوب)

## 🔐 Validation (التحقق):

### رقم الجوال:
- يجب أن يبدأ بـ `05`
- يتكون من 10 أرقام
- Pattern: `^05[0-9]{8}$`

### رقم الآيبان (للمسوقين):
- يجب أن يبدأ بـ `SA`
- يتكون من 24 حرف
- Pattern: `^SA[0-9]{22}$`

### كلمة المرور:
- 8 أحرف على الأقل
- يجب أن تتطابق مع تأكيد كلمة المرور

## 📝 للـ Backend Developer:

### API Register المحدث:

```typescript
POST /api/auth/register
Body: {
  // Common fields
  mobile: string,           // مطلوب
  email?: string,           // مطلوب للأطباء/التجار/المسوقين، اختياري للمستخدمين
  password: string,         // مطلوب
  confirmPassword: string,  // مطلوب
  userType: 'user' | 'doctor' | 'merchant' | 'affiliate',
  
  // Doctor-specific
  licenseNumber?: string,   // مطلوب للأطباء
  specialization?: string,  // مطلوب للأطباء
  
  // Merchant-specific
  commercialRegister?: string,  // مطلوب للتجار
  businessType?: string,        // مطلوب للتجار
  
  // Affiliate-specific
  bankName?: string,        // مطلوب للمسوقين
  iban?: string,            // مطلوب للمسوقين
}

Response: {
  success: boolean,
  message: string,
  userId: string,
  userType: string
}
```
